## 一、项目定位

本模组作为**整合包核心模组**开发，机械动力为**硬前置**。紫铜（Cuprite）被定义为**亚特兰蒂斯工业基石**，类比机械动力的安山合金——本身不造武器，但整个文明的机器、能源、管道、建筑都依赖它运转。

### 1.1 两条材料线（严格分离）

| 材料 | 生态定位 | 核心用途 | 世界观对应 |
|------|---------|---------|-----------|
| **紫铜** | 工业基石 | 机器、能源、管道、建筑框架 | 亚特兰蒂斯科技文明的血液 |
| **山铜** | 王权/战斗 | 武器、盔甲、神力、王权碎片 | 波塞冬祝福的神话金属 |

### 1.2 无Create不运行

由于机械动力为硬前置，所有紫铜加工、机器传动、流体网络直接依赖Create的API实现，无需维护独立的降级路径。这大幅简化代码结构，允许深度集成Create的动能、流体、装配系统。

## 三、紫铜工业体系

### 3.1 紫铜加工链（Create原生集成）

```
粗紫铜 → [熔炉/烟熏炉] → 紫铜锭
  ↓
紫铜锭 → [Create压板机] → 紫铜板
  ↓
紫铜板 → 大（小）齿轮+8*紫铜板 → 紫铜大（小）齿轮
  ↓
紫铜大（小）齿轮 → 紫铜大齿轮 + 山铜板 + 紫铜小齿轮 → 紫铜精密核心
```

### 3.2 紫铜工业材料清单

| 材料 | 注册名 | 来源 | 核心用途 |
|------|--------|------|---------|
| 紫铜板 | `cuprite_sheet` | 紫铜锭 → Create压板机 | 原材料|
| 山铜板 | `orichalcum_sheet` | 山铜锭 → Create压板机 | 原材料 |
| 紫铜大（小）齿轮 | `cuprite_gear` | 8*紫铜板 + 大（小）齿轮 → 工作台紫铜板包围齿轮 | 机械传动、机器零件 |
| 紫铜管道 | `cuprite_pipe` | 3×紫铜板 + 1×固化橡胶 → 序列配装/工作台菱形无序摆放 | 流体输送、装饰管道 |
| 紫铜机壳 | `cuprite_casing` | 1×紫铜锭 + 1×任意去皮原木 → 序列配装/工作台无序摆放 | 所有亚特兰蒂斯机器的外壳 |
| 紫铜精密核心 | `cuprite_？？？` | 紫铜大齿轮 + 山铜板 + 紫铜小齿轮 → 序列配装10次80%成功率，废料为紫铜锭或山铜板或紫铜大（小）齿轮 | 机器外壳、管道 |
### 3.3 固化橡胶（已有材料整合）

```
橡胶（rubber）+ 硫磺粉（brimstone_dust）→ [点金反应炉] → 固化橡胶板（vulcanized_rubber）
  ↓
用途：紫铜管道密封、潜水设备绝缘层、机器密封垫
```

## 四、Create联动机器（详细规格）

### 4.1 能源产生：紫铜涡轮机（Tidal Generator）

| 属性 | 规格 |
|------|------|
| 类型 | 动能产生器（继承 Create KineticBlock） |
| 转速 | 固定 256 RPM（不可调速） |
| SU输出 | 64su*RPM |
| 工作条件 | 完全浸入水中；Y 坐标；不可朝上（下）放置；可含水且含水后水不会流出方块|
| 深海加成 | Y < 60 时 SU 输出 +100%；Y < 0 时 SU 输出 +300% |
| 旋转方向 | 右键并手持扳手长按可以调整旋转方向 |
| 外观 | 紫铜机壳外壳，工作时中心有旋转涡轮 |
| 动画 | 涡轮随水流旋转，水粒子从管道喷出 |

**合成配方**：
```
[Create装配台]
紫铜机壳 ×1 + 紫铜小齿轮 ×2 + 山铜锭 ×2 + 固化橡胶 ×1 + 紫铜精密核心 ×1 → 序列配装100%成功率
→ 紫铜涡轮机 ×1
```

### 4.2 传动部件：防水轴承（Abyssal Bearing）

| 属性 | 规格 |
|------|------|
| 类型 | 继承 Create BearingBlock |
| 特殊能力 | 在水中不会生锈/损坏；普通Create轴承在水中长期使用会损坏（72000tick） |
| 外观 | 紫铜镶边的轴承，中心有防水密封垫 |

### 4.3 传动部件：紫铜传动轴（Cuprite Shaft）

| 属性 | 规格 |
|------|------|
| 类型 | 继承 Create ShaftBlock |
| 特性 | 在水中不会生锈/损坏；普通Create传动轴在水中长期使用会损坏（72000tick） |
| 外观 | 紫铜色金属轴，带细微纹理 |

### 4.7 流体机器：动力蒸馏器（Mercury Distiller）

| 属性 | 规格 |
|------|------|
| 类型 | 动能消耗 + 流体输入输出 |
| 输入 | 液体+ SU |
| 输出 | 固体产物 |
配方：液态流珠 → 汞碎块 + 银粒；盐水 → 粗盐
| 速率 | 每 20 tick 处理 100mB → 固体产物 |
| 外观 | 紫铜机壳 + 玻璃蒸馏管 + 冷凝器 |
| 动画 | 玻璃管内有汞液体流动，底部有加热红光 |

### 4.10 加工机器：紫铜压板机（Cuprite Press）

| 属性 | 规格 |
|------|------|
| 类型 | 继承 Create MechanicalPressBlock |
| 特性 | 专用于紫铜合金/紫铜锭，效率比通用压板机高 50% |
| 外观 | 紫铜机壳 + 压板机结构 |

## 十一、关键设计决策记录

### 11.1 紫铜与Create铜的关系

- Create原版有铜板/铜齿轮，但紫铜是**独立材料线**。
- 紫铜的加工效率更高（专用机器加成），且与亚特兰蒂斯主题绑定。
- 紫铜矿石不能直接用原版铜的配方，保持独特性。
- 紫铜与Create铜在机器中可以互换用于某些通用配方（如导线），但紫铜专用机器不可替代。

### 11.2 为什么不做独立动力网络

- 机械动力为硬前置，无需维护独立降级路径。
- 使用Create原生动能系统可保证与Create生态的无缝兼容（应力计、转速控制器、离合器等均可接入）。
- 开发效率更高，可专注于内容而非基础设施。

### 11.3 无Create航空学的设计约束

- 不加入飞艇、气球、航空引擎等飞行相关内容。
- Create的火车/轨道系统可以使用（用于海底隧道运输）。
- 所有机器设计围绕"水下/深海"主题，不触及空中内容。

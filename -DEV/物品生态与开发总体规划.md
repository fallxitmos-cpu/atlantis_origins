# Atlantis Origins：物品生态与开发总体规划

> 生成时间：2026-06-21  
> 对应版本：`0.2.5-alpha-fix6`  
> 用途：整合世界观、未来开发路线、物品生态扩展  
> 核心约束：**紫铜（Cuprite）为工业基础材料，不直接构成任何武器、工具与装备**。所有武器/装备/工具的刃部、主体必须使用山铜（Orichalcum）或其他战斗材料。紫铜仅作为合成配方中的辅助组件（框架、接点、柄、关节）。

## 一、故事大纲：从深渊到觉醒

### 1.1 核心命题

玩家不是冒险家，而是被波塞冬选中的**净化者**。亚特兰蒂斯沉没于深海，但真正的牢笼是神民们被自身罪恶固化的意志。所有战斗皆为意志交锋，所有怪物皆为执念投影。唯有击穿七种原罪，方能承载文明重量，成为被认可的新主神。

> **一句话**：武器可以杀死幻影，但无法净化心灵。真正的主神不是最强战士，而是第一个放下剑的人。

### 1.2 主线章节结构（九幕剧）

```
第一幕：溺入
  └─ 玩家首次完全浸入海洋，触发波塞冬的注视
  └─ 获得"氧气瓶"，了解最基本的生存法则
  └─ 遭遇深渊水母——不是敌人，是文明的呢喃
  └─ 解锁成就：真舒坦不是吗？/潜水新手

第二幕：压迫
  └─ 下潜至 Y<45，首次遭遇"压迫"状态（暗角、血丝）
  └─ 发现紫铜矿与山铜矿，建立材料认知（紫铜=工业/山铜=战斗）
  └─ 制作初学者潜水套装与泳镜，掌握基础装备体系
  └─ 发现发光鱼群、深海鳗鱼等被动生态，建立"深海不是空无一物"印象
  └─ 解锁成就：海洋，不过如此 / 点亮深渊

第三幕：工业曙光
  └─ 建立紫铜武器台（5×5），制作山铜工具与盔甲
  └─ 建造点金反应炉，掌握紫铜锭/山铜锭的炼制流程
  └─ 遭遇过压/超压，理解压强系统与深海威胁的层次性
  └─ 在深海守护者（Y<30）的袭击中首次面对真正的敌意
  └─ 解锁成就：深海守护者杀手 / 深渊恐惧

第四幕：原罪觉醒（0.3.x 主线开启）
  └─ 抵达亚特兰蒂斯外围废墟，触发"傲慢之影"投影
  └─ 获得第一块王权碎片：阿特拉斯之王的冠冕碎片
  └─ 理解核心叙事：怪物不是敌人，是被囚禁的神民意志
  └─ 解锁七宗罪试炼链

第五幕：镜像与怒火
  └─ 探索嫉妒区域（镜像大厅），面对"第十位王"
  └─ 遭遇愤怒之Boss"海神的利刃"，首个"不想杀但不得不打"的情感转折
  └─ 集齐第二、三块王权碎片
  └─ 深海守护者在塞壬之母净化前始终敌对，但行为暗示其悲剧性

第六幕：沉睡与贪欲
  └─ 探索懒惰区域（沉睡的城邦），Boss战在体内解谜
  └─ 面对贪婪之Boss"山铜之胃"，理解"吞噬"与"被吞噬"的双向恐惧
  └─ 获得纯净山铜矿脉，解锁顶级装备材料
  └─ 集齐第四、五块王权碎片

第七幕：吞噬与融合
  └─ 被"海渊之喉"吞噬，从内部战斗至核心
  └─ 面对色欲之Boss"塞壬之母"，最诡异、最私密的挑战
  └─ 塞壬之母净化后，深海守护者从敌对变为中立
  └─ 集齐第六、七块王权碎片

第八幕：终极审判
  └─ 七块碎片自动拼合为"Atlantis之心"
  └─ 在波塞冬神殿面对"亚特兰蒂斯之影"——玩家自身的完美倒影
  └─ 核心机制：碎片逐一飞出束缚Boss，玩家同时失去能力，战斗的本质是"承受平凡"
  └─ 最终抉择：终结（获得王冠，力量，但神民永不真正原谅自己）或宽恕（获得"Atlantis之忆"，无属性加成，但神民微笑）

第九幕：复兴（真结局后）
  └─ 亚特兰蒂斯从废墟苏醒，NPC建立日常作息与对话
  └─ 开放城市管理、NPC任务、庆典活动
  └─ 外围种植海藻农场、第一道城墙内恢复集市、第二道城墙内学者研究
  └─ 融合种（原深海守护者）建立独立社区
  └─ 波塞冬神殿成为主神殿，可举行仪式、接受祝福
```

### 1.3 叙事节奏与情感曲线

| 阶段 | 玩家情感 | 设计目标 |
|------|---------|---------|
| 初入海洋（Y>45） | 好奇、紧张 | 建立深海神秘氛围，水母群落安抚 |
| 压迫阶段（Y30-45） | 不安、探索欲 | 系统教学期，让玩家建立生存基础 |
| 过压/超压（Y<30） | 恐惧、压迫感 | 深海真正成为"敌人"，装备驱动成长 |
| 遭遇深海守护者 | 困惑、敌意 | 建立战斗习惯，但行为暗示非普通怪物 |
| 第一/二Boss | 成就感、同情 | 看到Boss真身后的解脱，"他们被困住了" |
| 海神利刃 | 悲壮、冲突感 | 第一个"不想杀"的情感转折点 |
| 塞壬之母 | 不适、反思 | 最诡异Boss，挑战对"美/进化"的认知 |
| 亚特兰蒂斯之影 | 恐惧、自我怀疑 | 面对完美自己，经历"失去能力"的恐慌 |
| 最终选择 | 沉默、觉悟 | 放下武器的5秒，是整个模组最长的5秒 |
| 真结局 | 温暖、平静 | 神民微笑，听到完整故事，参与重建 |
| 假结局 | 空虚、权力感 | 拥有王冠，但神民鞠躬时不笑 |

## 三、植物生态大纲：从浅海到深渊的植被体系

### 3.1 设计理念

深海植物不是装饰，而是**生态系统的基础**和**叙事的一部分**。不同深度的植物暗示着亚特兰蒂斯不同区域的历史痕迹：
- 浅海/外围：野生植物，与原版生态衔接
- 中环/深海带：受亚特兰蒂斯农业遗迹影响的变异植物
- 深渊/超深渊：与神民意志融合的发光/异变植物

### 3.4 植物合成与加工链

```
深渊水草 → 工作台 → 海藻卷（+面包）
深渊水草 → 烟熏炉 → 干海藻（不腐坏食物，2饥饿）
磷光藻粉 + 深海水瓶 → 酿造台 → 荧光药水（身体发光）
珊瑚苔 + 骨粉 → 催化为大型珊瑚簇（装饰方块）
藤壶壳 + 粗盐 → 腌制材料（用于食物加工）
```

### 3.5 世界生成配置

- **深渊水草**：替换原版海带生成于 `#minecraft:is_ocean`，但仅在 Y<60 生成，密度为原版的 60%
- **珊瑚苔**：与珊瑚礁结构一同生成，附加于 `seagrass` 放置步骤
- **磷光藻**：新增 Feature，仅在 Y<30 的 `deep_ocean` 群系生成，贴地生长
- **藤壶藤**：附加于沉船/海底遗迹的结构生成，仅生成于木质/石质墙面
- **记忆珊瑚**：仅在特定叙事区域（如嫉妒区域边缘）通过 StructureFeature 生成

## 七、武器大纲：七宗罪主题武器体系

### 7.1 设计理念

武器不是简单的数值升级，而是**与七宗罪主题绑定的叙事工具**。每件武器对应一种罪恶的"反面"——不是用罪恶对抗罪恶，而是用理解去化解。

> **核心规则**：紫铜不直接制作任何武器。所有武器的刃部/主体使用山铜或其他战斗材料，紫铜仅作为框架/接点/柄的辅助材料。

### 7.3 七宗罪主题武器（规划）

#### 7.3.1 傲慢 — 谦逊之刃（Modest Blade）

| 属性 | 说明 |
|------|------|
| **注册名** | `modest_blade` |
| **材料** | 山铜锭 × 3（刃）+ 紫铜机壳 × 1（护手）+ 山铜棍 × 1（柄） |

| **特殊机制** | 对"傲慢系"敌人（阿特拉斯之影、黄金幻影战士）攻击时，不会触发敌人的反击/降维压制机制 |

### 8.1 材料等级体系

```
第一层：基础矿物
  ├─ 紫铜（Cuprite）── 工业基础，不直接做装备/武器/工具
  ├─ 山铜（Orichalcum）── 王权/战斗，顶级装备/武器/工具
  ├─ 银（Silver）── 神圣/净化，仪式/特殊武器/面甲
  ├─ 硫磺（Brimstone）── 化工基础，橡胶/药剂
  ├─ 盐（Salt）── 食品/防腐/建筑
  └─ 煤/炭黑（Carbon Black）── 能源/化工

第二层：工业中间品
  ├─ 紫铜板/齿轮/线/管道/机壳/框架/线圈 ── 紫铜加工产物（工作台/紫铜武器台合成）
  ├─ 山铜板 ── 山铜加工产物
  ├─ 固化橡胶（Vulcanized Rubber）── 橡胶+硫磺，密封/绝缘
  ├─ 银汞齐合金（Silver Amalgam）── 银+汞，特殊合金
  └─ 山铜棍（Orichalcum Stick）── 山铜工具手柄

第三层：Boss/叙事材料
  ├─ 七块王权碎片 ── 不可掉落/不可交易，剧情推进
  ├─ 记忆碎片 ── 水母掉落，合成回忆瓶
  ├─ 痛苦结晶 ── 刺胞水母掉落，合成痛苦药剂
  ├─ 凝视核心 ── 凝视水母掉落，合成启示之眼
  ├─ 融合之鳞 ── 深海守护者掉落，合成形态稳定药剂
  └─ 海神血肉 ── 特殊食物，进化路线

第四层：合成装备/武器/工具
  ├─ 山铜工具四件套（镐/斧/锹/锄）
  ├─ 山铜盔甲四件套
  ├─ 山铜三叉戟
  ├─ 七宗罪主题武器（7件）
  ├─ 特殊工具（声呐/探测器/清除器等）
  └─ 终极武器（波塞冬的怜悯）
```

### 8.2 紫铜参与规则（强制）

紫铜（Cuprite）及其加工产物（板/齿轮/线/管道/机壳/框架/线圈）**不得**作为任何武器、工具、盔甲的**主体材料**或**刃部材料**。紫铜在配方中的角色仅限于：

1. **框架/外壳**：为武器/工具提供结构支撑（如三叉戟的接点、锄头的框架）
2. **能量传导**：紫铜线圈用于需要能量传导的设施（如清醒之杖、氧气充填站）
3. **密封/防水**：紫铜管道/机壳用于水下设施的密封结构
4. **工业加工**：所有机器方块（武器台、反应炉、蒸馏器等）的外壳必须使用紫铜机壳

### 8.3 山铜参与规则（强制）

山铜（Orichalcum）是**唯一的顶级战斗材料**。所有直接造成伤害的武器、提供防御的盔甲、用于挖掘/采集的工具，其主体/刃部/板甲部分必须使用山铜。山铜在配方中的角色：

1. **武器主体**：剑刃、斧刃、镐头、三叉戟刃部
2. **盔甲板甲**：头盔/胸甲/护腿/靴子的防护板
3. **工具头部**：镐头、锹头、锄头刃
4. **手柄/握柄**：山铜棍是所有山铜工具/武器的标配手柄
